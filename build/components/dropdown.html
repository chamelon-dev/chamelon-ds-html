Выпадающее меню

Компонент позволяет добавить в интерфейс компактный способ отображения нескольких вариантов, которые появляется при взаимодействии с элементом (например, значком или кнопкой).

Выпадающее меню, иконки + текст

Предназначен для создания выпадающего меню, в котором подписи к пунктам меню снабжены иконками. Компонент составной и содержит в себе компоненты иконок и кнопок

  • Посмотреть
  • Перейти к списку
  • Изменить
  • Удалить
  • Добавить в избранное
  • HTML
  • React
Исходный код этого компонента находится по пути sass/components/_dropdown.scss
