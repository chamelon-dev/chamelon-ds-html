Шапка

Компонент предназначен для создания верхнего колонтитула страницы. Обычно содержит логотип, навигационное меню или ссылки на связанные документы.

Шапка, фиксированный макет

Пример шапки для фиксированных центрированных шаблонов. Компонент составной и содержит в себе компоненты логотипа, кнопок, иконок и навигационного меню
