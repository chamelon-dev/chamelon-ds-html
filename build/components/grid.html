Сетка

Гибкая сетка, для создания макетов всех форм и размеров благодаря системе из двенадцати столбцов и семи точек останова

Общие правила

Общие правила при построении сеток

1. Все колонки (сколько бы их не существовало на странице) выравниваются по горизонтали и имеют одинаковую высоту

2. Колонки (элементы с классом .col-* и .col / компонент <Column />) должны являться прямыми потомками строк. Не допускается использование колонок вне строк

3. Строки (элементы с классом .row / компонент <Row />) должны являться прямыми потомками контейнера. Не допускается использование строк вне контейнера

4. Контейнеры (элементы с классом .container / компонент <Grid /> для фиксированных центрированных макетов и элементы с классом .container-fluid / компонент <Grid /> со значением true свойства fluid для резиновых макетов и резиновых макетов с сайдбаром) являются прямыми родителями всей сетки. Не допускается использование контейнера внутри контейнера

5. Допускается вкладывание в колонки других колонок. Добавьте новый .row / компонент <Row /> и набор .col-* столбцов / компонент <Column /> в существующую .col-* колонку / компонент <Column />. Вложенные строки должны включать набор колонок, которые в сумме составляют до 12 или меньше.

Параметры сетки

Система сеток может адаптироваться ко всем семи точкам останова по умолчанию. Семь уровней сетки по умолчанию следующие:

  • Очень маленький (xs)
  • Маленький (sm)
  • Средний (md)
  • Большой (lg)
  • Очень большой (xl)
  • Очень очень большой (xxl)
  • Очень очень большой (xxxl)

Каждая из этих точек останова имеет собственный уникальный префикс класса. Вот как меняется сетка в этих точках останова:
