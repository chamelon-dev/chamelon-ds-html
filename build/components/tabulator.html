Табулатор

Компонент позволяет создавать интерактивные таблицы за секунды из любой таблицы HTML, массива JavaScript, источника данных AJAX или данных в формате JSON.
Дополнительная информация - http://tabulator.info/

Пример использования интерактивной таблицы, построенной из JSON-объекта

  • HTML
  • React
Исходный код этого компонента находится по пути sass/components/_tabulator.scss
СвойствоТипОбязательноПо-умолчаниюОписание
dataМассив объектовДаМассив объектов,содержащих строки и ячейки
columnsМассив объектовДаМассив объектов, содержащих название колонок
paginationSizeЧислоНет10Количество строк на каждой странице
resizableColumnsЛогическийНетfalseРазрешить изменять размер столбцов
layoutСтрокаНетfitDataFillРасположение столбцов в таблице

fitData - подогнать столбцы к данным
fitDataFill - подобрать столбцы по размеру данных и заполнить
fitDataStretch - подогнать столбцы к данным и растянуть последний столбец
fitDataTable - подогнать таблицу и столбцы к данным
fitColumns - подобрать столбцы по ширине таблицы
Исходный код этого компонента находится по пути sass/components/_tabulator.scss и components/Tabulator.js
