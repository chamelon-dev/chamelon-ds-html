Навигационное меню

Компонент предназначен для добавления в интерфейс навигационных меню различной глубины вложенности и направления.

Общие правила

Общие правила при работе с навигационными меню

Пункты меню верхнего уровня должны содержать смысловую иконку. В случае отсутствия таковой, рекомендуется использовать иконку «page» из компонента иконок, группы «Иллюстрации». Эти иконки должны находиться перед текстовым названием пункта меню и иметь класс .menu-prefix

Все текстовые названия пунктов меню и подменю должны быть обернуты в элемент«span class="menu-label"»...«/span»

Пункты подменю, содержащие в себе другие подменю, должны содержать иконку «chevron-right» (для фиксированных меню) / «chevron-down» (для горизонтальных и вертикальных меню) из компонента иконок, группы «Иконки форм». Эта иконка должна находиться после текстового названия пункта меню и иметь класс .menu-postfix

Блоки, содержащие подменю, должны быть обернуты в элемент<div class="menu-submenu">...</div>

Ссылки, которые не осуществляют переход на какие-либо страницы, должны содержать атрибут data-prevent для предотвращения поведения по-умолчанию

Элемент, который содержит в себе подменю и является прямым родителем для него, в обязательном порядке должен содержать атрибут tabindex="0"для корректной работы в браузере Safari

Использование предваряющих иконок в пунктах подменю для горизонтального меню не допускается

Глубина вложенности для всех типов меню (кроме фиксированного меню) ограничена лишь физическими размерами устройства и фантазией разработчика, но в зависимости от ситуации рекомендуется использовать не более трех уровней вложенности

По-умолчанию, ширина выпадающего меню равна 240 пикс., для увеличения ширины до 400 пикс. используется дополнительный класс .menu-submenu-wide, который должен быть примененн к блоку <div class="menu-submenu">...</div>

Фиксированные меню предназначены для использования совместно с компонентом Сайдбара. Использование этого типа меню во всех других случаях недопустимо

Фиксированное меню

Пример простейшего фиксированного меню с тремя уровнями вложенности и активным пунктом меню. В этом типе меню вложенные подменю растягиваются на всю высоту родителя, фиксируются слева и используют автоматическую прокрутку, если пунктов меню во вложенных меню больше, чем позволяет высота контейнера.
