Уведомления

Компонент предназначен для добавления в интерфейс сообщений, предназначенных для информирования пользователей о важных действиях.

Общие правила

Общие правила при работе с уведомлениями

Количество видимых уведомлений не должно превышать 5

Отступы слева и справа от кнопку раскрытия уведомлений должны составлять не менее 16 пикс. Для установки отступов рекомендуется применять CSS-утилиты внешних отступов
