Следующие параметры могут быть применены к каждому узлу дерева

ОбъектОписаниеПример
{"opened": true / false}Раскрывает узел при наличии дочерних узлов при инициализации плагина.data-jstree='{ "opened" : true }'
data-jstree='{ "opened" : false }'
{"selected": true / false}Выбирает узел и отмечает как "выбранный".
Если применяется к дочернему узлу, все его родительские узлы будут раскрыты при инициализации плагина.		data-jstree='{ "selected" : true }'
data-jstree='{ "selected" : false }'
{"disabled": true / false}Запрещает выбор узлаdata-jstree='{ "disabled" : true }'
data-jstree='{ "disabled" : false }'
Исходный код этого компонента находится по пути sass/components/_jstree.scss

AJAX-дерево

Пример многоуровневого раскрывающегося дерева на основе данных, получаемых с сервера

  • HTML
  • React
data-атрибутОписание
data-jstree-ajaxУказывает, что дерево будет построено на основе данных с сервера
data-url-rootСсылка на JSON-ресурс, которая возвращает список всех родителей
data-url-childrenСсылка на JSON-ресурс, которая возвращает список всех детей
По-умолчанию, к этой ссылке будет добавлен GET-параметр идентификатора родителя
