Тестовые поля

Компонент предназначен для добавления в интерфейс различных текстовых полей.

Общие правила

Общие правила при работе с текстовыми полями

Все поля в обязательном порядке должны содержать текст-заполнитель (атрибут placeholder)

Если отсутствует возможность указать смысловой текст-заполнитель, то он должен совпадать с текстовой меткой поля (элемент label)

Если обстоятельствами не предусмотрено наличие текста-заполнителя, в качестве его значения используется пробел

Все поля в обязательном порядке должны быть обернуты в блок с классом .field. К нему же должны применяться CSS-утилиты

Кнопка очистки поля не совместима с иконками валидации
