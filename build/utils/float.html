Обтекание

Определяет, по какой стороне будет выравниваться элемент, при этом остальные элементы будут обтекать его с других сторон

Обтекание

Позволяет задать или изменить обтекание (float) у элементов

Обтекание задается элементам путем присвоения класса вида float-{type}

{type}ПримерОписание
none<div class="float-none"></div>Обтекание элемента не задаётся
left<div class="float-left"></div>Выравнивает элемент по левому краю, а все остальные элементы, вроде текста, обтекают его по правой стороне
right<div class="float-right"></div>Выравнивает элемент по правому краю, а все остальные элементы обтекают его по левой стороне
Исходный код этого компонента находится по пути sass/uils/_float.scss
