Отступы

Добавление внутренних и внешних отступов к различным элементов интерфейса

Внешние отступы

Позволяют задать внешние (margin) отступы элементам

Внешние отступы задаются элементам путем присвоения класса вида m{sides}-{size}

{sides}{size}ПримерОписание
(пусто)Любое целое число от 4 до 160, кратное 4<div class="m-32"></div>Элементу будет присвоен внешний отступ размером 32px со всех сторон
t<div class="mt-32"></div>Элементу будет присвоен внешний отступ размером 32px сверху
r<div class="mr-32"></div>Элементу будет присвоен внешний отступ размером 32px справа
b<div class="mb-32"></div>Элементу будет присвоен внешний отступ размером 32px снизу
l<div class="ml-32"></div>Элементу будет присвоен внешний отступ размером 32px слева
v<div class="mv-32"></div>Элементу будет присвоен внешний отступ размером 32px сверху и снизу
h<div class="mh-32"></div>Элементу будет присвоен внешний отступ размером 32px слева и справа
Исходный код этого компонента находится по пути sass/uils/_spacing.scss

Внешние отступы

Позволяют задать внутренние (padding) отступы элементам

Внутренние отступы задаются элементам путем присвоения класса вида p{sides}-{size}

{sides}{size}ПримерОписание
(пусто)Любое целое число от 4 до 160, кратное 4<div class="p-32"></div>Элементу будет присвоен внутренний отступ размером 32px со всех сторон
t<div class="pt-32"></div>Элементу будет присвоен внутренний отступ размером 32px сверху
r<div class="pr-32"></div>Элементу будет присвоен внутренний отступ размером 32px справа
b<div class="pb-32"></div>Элементу будет присвоен внутренний отступ размером 32px снизу
l<div class="pl-32"></div>Элементу будет присвоен внутренний отступ размером 32px слева
v<div class="pv-32"></div>Элементу будет присвоен внутренний отступ размером 32px сверху и снизу
h<div class="ph-32"></div>Элементу будет присвоен внутренний отступ размером 32px слева и справа
Исходный код этого компонента находится по пути sass/uils/_spacing.scss
